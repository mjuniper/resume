          Presentations

            I thought it would be clever to create my resume as a javascript object. Then I started thinking that it would be cool to put some unit tests on it to validate it. The tests are kind of token but they end up being another way to present the data. Then I started thinking that people might not think it was all that clever and would rather see a nicely formatted resume. So (since I've always said that any web developer who's worth anything should have an HTML resume) I built this page which uses Twitter Bootstrap for the layout and generates the content from the Javascript object using ICanHaz & Mustache templates. I thought about using Backbone.js for this but that seemed like overkill.

            Don't worry, there's a print stylesheet so you should be able to print it, or print it to a pdf.
